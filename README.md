# Densil Vincent — Portfolio

Personal portfolio site for **Densil Vincent** — Certified PMP, Scrum Master, ITIL Strategist, and Lean Six Sigma practitioner with 15+ years delivering excellence in IT, fintech, and e-commerce projects, customer support, and operations.

🔗 **Live site:** https://densilvincent.github.io

## About

A single-page, responsive portfolio built as one self-contained `index.html` file — no build step, no dependencies to install. It showcases professional experience, skills, certifications, key achievements, and contact details.

## Built with

- Plain HTML & CSS (CSS custom properties for theming)
- Vanilla JavaScript (scroll-reveal animations, adaptive navigation)
- [Google Fonts](https://fonts.google.com/) — Fraunces, IBM Plex Sans & Mono
- [logo.dev](https://logo.dev) — certification issuer logos

## Hosting

Served via **GitHub Pages** from the `main` branch (root folder).

### Updating the site

Edit `index.html`, then either:

- **Browser:** upload the new file over the old one and commit, or
- **Git:**
  ```bash
  git add index.html
  git commit -m "Update portfolio"
  git push
  ```

Changes go live within a minute or two.
